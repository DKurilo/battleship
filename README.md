# Battleship
Pure.. Functional..  
Haskell + MongoDB + TypeScript + React  
In progress..  
Seeing this can be dangerous. Too in progress.  

To build client:  
```
yarn install
yarn build
```
  
To build server:  
Copy env.template to .env. Fix parameters.  
```
cabal install
cabal run --p 9000
```
  
In browser: `http://localhost:9000/`
